# Harbour Line Advisory

Private landing page for [harbourlineadvisory.com](https://harbourlineadvisory.com).

## Public footprint

The public site is deliberately minimal. It contains no description of services, sectors, clients, projects or commercial activity.

## Structure

- `index.html` — public holding page
- `styles.css` — visual system
- `access.css` — access-dialog styling
- `script.js` — temporary access routing
- `favicon.svg` — site icon
- `CNAME` — custom domain
- `access-system/` — optional private-access service
- `docs/ACCESS-CONTROL.md` — technical deployment notes

## Deployment

GitHub Pages publishes from the root of the `main` branch.

## Privacy

Operational records, submissions and credentials are maintained outside this public repository. Secrets, plain access codes and private records must never be committed here.
